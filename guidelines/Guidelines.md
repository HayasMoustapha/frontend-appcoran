# Guidelines — Règles simples du projet

Ce document sert à garder une interface cohérente et facile à maintenir.

## 1) Règles de design (simples)
- Couleurs : toujours utiliser la palette du thème Nuit Céleste.
- Texte : toujours lisible (contraste élevé).
- Cartes : même style partout (padding, bordures, hover).
- Boutons : un bouton principal par section.

## 2) Règles UX
- Éviter les actions cachées : un bouton doit être visible et clair.
- Donner un retour utilisateur (snackbar ou message) après une action.
- Sur mobile : tout doit rester cliquable sans zoom.

## 3) Règles code
- Pas de logique complexe directement dans le JSX.
- Toujours vérifier si une donnée existe avant de l’afficher.
- Préférer des composants réutilisables.

## 4) Résultat attendu
Une interface harmonieuse, agréable, et stable sur tous les appareils.
